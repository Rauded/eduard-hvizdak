# eduardhvizdak.com

Source of [eduardhvizdak.com](https://eduardhvizdak.com), the personal site of Eduard Hvizdak: AI engineer and indie founder.

Create React App, TypeScript, SCSS. Deploys to Vercel on every push to `main`.

## Local dev

```
npm install
npm start
```

Production build:

```
CI=false npm run build
```

CRA treats warnings as errors when `CI` is set; keep the flag.
